Description of the model names

Original Audio [Code]: The original audio recording from a speaker with cerebral palsy. The code in parenthesis is the speaker - F: Female, M: Male, with the number representing the speaker from that catagory

Baseline LibriTTS: A TTS model trained from scratch on LibriTTS data only

Finetuned Block: The Baseline LibriTTS model finetuned for many-to-one voice conversion on UASpeech data. The training setup splits the train, test and dev data via recording blocks, which results in word overlap between training and testing

Finetuned Utterance: The Baseline LibriTTS model finetuned for many-to-one voice conversion on UASpeech data. The training setup splits the train, test and dev data on an utterance level, meaning there will be no word overlap between sets
